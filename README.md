## 1. Introduction to Information Technology
Information Technology (IT) refers to the use of computers, software, networks, and electronic
systems to store, process, and transmit information efficiently.

## 2. Hardware and Software
Hardware includes physical components such as CPU, memory, keyboard, and monitor.
Software refers to programs and applications that instruct the hardware to perform tasks.
Software is classified into system software and application software.

## 3. Operating System
An Operating System (OS) acts as an interface between the user and computer hardware.
It manages processes, memory, files, and input/output devices.

## 4. Computer Memory
Memory is used to store data and instructions.
It is classified into primary memory (RAM, ROM) and secondary memory (HDD, SSD).

## 5. Input and Output Devices
Input devices are used to enter data into a computer (keyboard, mouse).
Output devices display processed information (monitor, printer).
